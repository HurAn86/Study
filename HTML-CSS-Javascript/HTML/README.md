# HTML
## block and inline
### Block (have space / start at new line):
- div 
- h1
- p
- section
- ...
### inline (no space):
- span 
- button
- input
- ...
  
##  Each page can have 3 div:
- header
- body
- footer

## lists
  unorder list (ul) and order list(ol)
```HTML
<ul>
    <li>list1</li>
    <li>list2</li>
    <li>list2</li>
</ul>
```

## display
- use "$lt;" -- to display "<"
- use "$gt;" -- to display ">"
- use "$amp;" -- to display "&"
- use "$copy;" the next letter @
- none breaking space "$nbsp;"
- 不同的字体可能导致乱码: "$quot;"
