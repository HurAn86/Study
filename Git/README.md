# Git

## Introduction to Git
### Intialize
```sh
git init
```
### Clone the remote repository to local 
```sh
git clone https://github.com/HurAn86/projectname.git
```
### Commit
```sh
git add -A && git commit -m "..."
```
### Push
```sh
git push 
```
### Others
```sh
git checkout master
git checkout <hash>
git checkout -b <branch_name>
```

###删除account的历史记录
```sh
git credential-osxkeychain erase
```
